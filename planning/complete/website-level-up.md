# Website Level Up

The website needs to have all of the same content, functionality and features.

But I'd like it to be a lot more professional, interesting, engaging and fun.

You can change whatever you want about the website as long as it keeps what I mentioned above. Meaning we can't have a
situation where download links are missing or broken or docs are broken or missing, etc.

But other than that, it needs to be modernized and so on.

Stick to the same technology. It should be an astro site.

You can add things, but stay away from crazy things like heavy CSS frameworks and such.

Build out your proposal and we will iterate from there, together.

Do not delete the above summary.

Track your progress below.

## Proposal

Modernize the Astro website by keeping every current route, download URL, documentation entry point, analytics hook, and social/footer link intact while replacing the homepage with a more polished product experience. The first pass should make the homepage feel clearer and more professional, with stronger visual hierarchy, better responsive behavior, and a more purposeful explanation of the Blender-to-Godot workflow.

# Codex Plan: FPE SEO Content Patch

## Core Idea

Folded Paper Engine is:

```txt
Instant Game Mechanics from Blender to Godot
```

FPE lets users author gameplay meaning inside Blender, export to Godot, and get working mechanics without rebuilding everything manually in the engine.

This SEO pass must reinforce that idea without damaging the existing site content, documentation, reference material, visual style, donation CTAs, or community tone.

## Primary Goal

Improve Google’s understanding of Folded Paper Engine for high-value non-branded queries while preserving the current site.

Target meaning:

```txt
Folded Paper Engine creates instant game mechanics from Blender-authored scenes exported to Godot.
```

Do not replace the site’s existing educational/reference content.
Do not rewrite major sections unless absolutely needed.
Do not create thin SEO pages.
Do not stuff keywords.
Do not mutate the docs into marketing copy.
Add only useful content that helps real users understand the platform.

# Implementation Rules

## Preserve existing content

Existing homepage sections, docs, guide content, API references, support CTAs, and platform explanations should remain intact unless a small wording adjustment is clearly beneficial.

Prefer:

```txt
Add a sentence.
Add a short paragraph.
Improve title/meta.
Add a contextual internal link.
```

Avoid:

```txt
Replace entire sections.
Rewrite docs into sales pages.
Create shallow keyword pages.
Change information architecture unnecessarily.
```

## Quality bar

Every added sentence must help a real person understand FPE better.

Added content should be:

* accurate
* specific
* useful
* consistent with the current voice
* community-minded
* respectful of artists, developers, educators, solo devs, and small teams

No SEO sludge.
